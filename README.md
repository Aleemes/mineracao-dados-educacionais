# mineracao-dados-educacionais
Scripts do TCC: Mineração de Dados Educacionais aplicada ao IDEB com CRISP-DM e SHAP

Scripts desenvolvidos para o Trabalho de Conclusão de Curso (TCC) 
do curso de Licenciatura em Computação da Universidade Federal do Paraná (UFPR).

## Sobre o trabalho
Este repositório contém os scripts utilizados para identificar quais variáveis 
do Censo Escolar exercem maior influência sobre o IDEB nas escolas 
públicas de Ensino Fundamental brasileiro, por meio do processo CRISP-DM e 
técnicas de Inteligência Artificial Explicável.

## Scripts
- **Script 1** — Pré-processamento e seleção de variáveis
- **Script 2** — Comparação de modelos de regressão com validação cruzada
- **Script 3** — Explicabilidade com SHAP e treinamento da rede neural MLP
